# Civitai Seen Tracker

Civitaiのモデル一覧画面で、過去に見たことのあるモデルをグレーアウトし、閲覧済みであることをわかりやすくするTampermonkey用ユーザースクリプトです。

## インストール方法
お使いのブラウザに [Tampermonkey](https://www.tampermonkey.net/) などをインストールした状態で、以下のリンクをクリックするとインストール（またはアップデート）画面が開きます。

👉 **[インストールはこちらをクリック](https://raw.githubusercontent.com/Tail954/CivitaiSeenTracker/master/civitai-seen-tracker.user.js)**

インストール後、スクリプトに変更があった場合はTampermonkeyが自動的にアップデートを検知して更新します。

## 概要
- スクロールして画面外に出たモデルを「閲覧済み」として記録します。
- 閲覧済みのモデルは透明度が下がり、白黒（グレースケール）になります。
- マウスオーバーすると元の表示に戻ります。

## キャッシュ（閲覧履歴）の保存場所とリセット方法

本スクリプトで記録された「閲覧済みモデル」のデータは、お使いのブラウザの **ローカルストレージ（localStorage）** に保存されています。
閲覧履歴をすべてリセット（削除）して、すべてのモデルのグレーアウトを解除したい場合は以下の手順を行ってください。

1. Civitaiのページ（ https://civitai.com/ ）を開きます。
2. キーボードの `F12` キー（または `Ctrl` + `Shift` + `I`）を押して、ブラウザの**開発者ツール**を開きます。
3. 上部のタブから **「Application（アプリケーション）」** または **「Storage（ストレージ）」** を選択します。
4. 左側のメニューの「Storage」から **「Local Storage」** を展開し、`https://civitai.com` をクリックします。
5. 右側に表示されるリストの中から **`civitai_seen_models`** というキー（Key）を探します。
6. 見つけたら、右クリックから「Delete（削除）」を選択するか、選択状態で `Delete` キーを押します。
7. ページを再読み込み（F5）すると、すべての履歴がリセットされます。
